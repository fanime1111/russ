Виды простых предложений

Теоретический материал

Простое предложение

Простое предложение — это предложение, в котором одна грамматическая основа (один предикативный центр).

Классификация простых предложений

1. По цели высказывания

  • Повествовательные — содержат сообщение: На улице идёт дождь.
  • Вопросительные — содержат вопрос: Ты придёшь завтра?
  • Побудительные — содержат побуждение (просьбу, приказ, совет): Принеси мне воды. Не опаздывай!

2. По эмоциональной окраске

  • Восклицательные — произносятся с особой интонацией, выражают сильные чувства: Как красиво!
  • Невосклицательные — нейтральная интонация: На улице красиво.

3. По наличию главных членов

  • Двусоставные — есть подлежащее и сказуемое: Птицы поют.
  • Односоставные — только один главный член:
    • Определённо-личные — сказуемое в форме 1 или 2 лица: Люблю грозу в начале мая. Идёшь по улице и радуешься.
    • Неопределённо-личные — сказуемое в форме 3 лица мн. числа или прош. времени мн. числа: В дверь стучат. Его пригласили на праздник.
    • Обобщённо-личные — действие относится к любому лицу (пословицы, афоризмы): Без труда не выловишь и рыбку из пруда. Цыплят по осени считают.
    • Безличные — нет и не может быть подлежащего: Светает. Мне холодно. Нужно уходить.
    • Назывные (номинативные) — только подлежащее: Ночь. Улица. Фонарь. Аптека.

4. По наличию второстепенных членов

  • Распространённые — есть второстепенные члены: Маленькая девочка читает интересную книгу.
  • Нераспространённые — только главные члены: Девочка читает.

5. По полноте структуры

  • Полные — все необходимые члены предложения присутствуют: Я читаю книгу.
  • Неполные — один или несколько членов пропущены, но понятны из контекста: — Что ты читаешь? — Книгу. (пропущено «Я читаю»)

6. По наличию осложняющих элементов

  • Осложнённые — содержат однородные члены, обособленные члены, вводные слова, обращения: Мама, папа и сестра пришли домой.
  • Неосложнённые — не содержат осложняющих элементов: Мама пришла домой.

Сводная схема классификации

Критерий Виды
По цели высказывания Повествовательное, вопросительное, побудительное
По эмоциональной окраске Восклицательное, невосклицательное
По наличию главных членов Двусоставное, односоставное
По наличию второстепенных членов Распространённое, нераспространённое
По полноте Полное, неполное
По наличию осложнения Осложнённое, неосложнённое
